Application de Gestion de Tâches
Cette application de gestion de tâches est développée en utilisant Kotlin et Jetpack Compose. Elle permet aux utilisateurs de créer, visualiser et gérer leurs tâches de manière intuitive et efficace.

Fonctionnalités
Affichage du tableau de bord avec les tâches à faire.
Création de nouvelles tâches.
Interface utilisateur conviviale grâce à Jetpack Compose.

Configuration requise
Android Studio 4.0 ou version ultérieure.
JDK 8 ou version ultérieure.
Un appareil Android ou un émulateur Android pour tester l'application.
Installation
Clonez ce dépôt sur votre machine locale :

bash
Copy code
git clone git@github.com:Sadako-amaryllis/mytasktd.git
Ouvrez le projet dans Android Studio.

Attendez que les dépendances soient téléchargées et synchronisées.

Exécutez l'application sur un émulateur Android ou un appareil Android connecté.
